강신범

끊임없이 성장하는 Android Developer

10년 이상의 실무 경험을 바탕으로 사용자 중심의 가치있는 서비스를 만듭니다. 안정적이고 확장 가능한 아키텍처 설계에 강점이 있으며, 새로운 기술을 학습하고 적용하는 것을 즐깁니다.

핵심 역량

Android 앱 개발 전문성

  • Kotlin, Java 기반 네이티브 앱 개발
  • MVVM, Multi-Module 등 최신 아키텍처 설계
  • Jetpack Compose를 활용한 선언형 UI 구현
  • Coroutine, Flow 기반의 능숙한 비동기 처리

신규 서비스 개발 및 고도화

  • 신규 FnB 솔루션(POS, Kiosk) 개발 총괄 및 런칭
  • 레거시 코드 리팩토링으로 서비스 안정성 30% 향상
  • 다양한 도메인(모빌리티, 금융, 공공) 서비스 개발 경험

시스템 및 장비 연동

  • Socket, REST API 기반 서버-클라이언트 연동
  • 외부 장비(로봇팔, 아두이노) SDK 및 JNI 연동
  • 다양한 지도 SDK(Naver, Kakao, Google) 활용

프로젝트 리딩 및 협업

  • 최대 6명의 팀 리딩 및 신규 앱 2건 성공적 런칭
  • 기획, 디자인, 서버 등 유관부서와 원활한 협업
  • Agile/Scrum 방법론 기반의 효율적인 프로젝트 관리
